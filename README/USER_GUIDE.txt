WHAT IS ANONYMIZER?
-------------------
Anonymizer is a tool for anonymizing MQXLIFF/TMX documents exported from memoQ.

================================================================================
                       ANONYMIZATION FILTERS
================================================================================

The program has 6 detection layers:

1. REGEX CLINICAL IDs
2. PRESIDIO (PII)
3. BIOMEDICAL ENTITIES
4. CUSTOM DICTIONARY
5. PROTECTED TERMS (whitelist)
6. SAFE REGEX (multilingual structured PII detection)

SAFE REGEX
----------
Independent anonymization layer using curated multilingual regex patterns.
Detects structured PII/PHI with very low false positives.

Supports:
- Emails
- Clinical IDs
- National IDs
- Financial data
- URLs & IPs
- Phone numbers
- Postal addresses
- Postal codes
- Names with titles
- Dates of birth

Fully language-agnostic (EN, ES, FR, DE, IT, PT, NL, ZH, JA)

7. DEDUPLICATE TMX SEGMENTS
---------------------------
(Post-processing filter, not a detection layer)

- Removes duplicate or similar segments
- Configurable similarity threshold
- Removed segments appear in Excel under "Excluded Segments"

Additionally:
- Inline XML tags (<bpt>, <ept>, etc.) are automatically removed
- TMX is cleaned for direct import into memoQ
